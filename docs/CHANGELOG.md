# 更新日志

## [2024-03-17] AI 智能命令交互增强

### 新增功能

#### Shell/Agent 模式切换
- 在终端标签页右侧添加模式切换开关
- Shell 模式：传统终端体验，专注命令执行
- Agent 模式：AI 增强模式，支持自然语言交互
- 快捷键 `Ctrl+Shift+I` 快速切换模式

#### AI 命令助手（Shell 模式）
- 可拖动的悬浮面板，按 `Ctrl+Shift+Y` 唤出
- 输入中文描述，AI 生成命令
- 显示命令解释
- 支持复制命令、插入到终端

#### Agent 模式 AI 交互
- 中文输入自动触发 AI 响应
- 显示风险等级标签（安全/中等/高危/极高危）
- 三种操作按钮：执行、修改、拒绝
- 命令执行后 AI 自动给出后续建议
- 支持开启新会话

#### 自定义终端实现
- 完全自定义的终端组件，替换 xterm.js
- 支持 ANSI 转义序列解析（颜色、样式）
- 支持 conda 环境提示符（如 `(base)`）
- 输入行集成在终端输出区域内

#### Tab 命令补全
- 按 Tab 键请求 AI 命令补全建议
- 显示命令列表和说明
- 键盘导航：↑↓ 选择，Tab/Enter 接受，Esc 关闭

### 改进

- 优化终端输出显示，正确处理 OSC 控制序列
- 优化提示符提取，支持多种格式（包括 conda 环境）
- 移除自动触发命令提示，改为 Tab 键主动触发
- 改善 WebSocket 连接稳定性，防止重复连接

### 技术细节

- 新增 `CustomTerminal.tsx` 自定义终端组件
- 新增 `CommandAssistant.tsx` AI 命令助手组件
- 新增 `ansiParser.ts` ANSI 转义序列解析工具
- 更新 `appStore.ts` 添加终端模式状态管理
- 更新 `TerminalArea.tsx` 添加模式切换 UI
